# Initial Setup

## Onboarding
After launching Dione for the first time, you'll be greeted with the onboarding screen. Let's get you started!

![Onboarding](/images/onboarding.png)

## 1. Login (Optional)

Logging in is **recommended** to access all features, but you can skip this step if you prefer.

1.  Click **Login** to connect via Google, Discord, or GitHub.
2.  Authenticate in your browser.
3.  Click **Open Dione** when prompted.

**Note**: We take your privacy seriously. Read our [Privacy Policy](https://getdione-app.deeivihh.workers.dev/legal/privacy-policy).

![Login](/images/login.png)

*Having trouble? Check the [Troubleshooting Guide](/troubleshooting/common-issues#login-button-doesnt-work).*

---

## 2. Select Installation Path

Choose a folder where Dione will download apps and dependencies (creating `apps` and `bin` folders).

**⚠️ Important**: Do NOT select the same folder where `dione.exe` is located. Pick a separate directory to avoid update conflicts.

![Select installation path](/images/onboarding2.png)

---

## 3. All Set!

You are now ready to explore Dione.

To restart this process later, go to **Settings -> Reset Onboarding**.

![Onboarding Finished](/images/onboarding-finished.png)
