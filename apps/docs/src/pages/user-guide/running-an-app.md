# Running an App

## 1. Start

Navigate to the **Actions** screen and click **Start**.

![Actions screen](/images/actions-screen.png)

## 2. Monitor & Control

The live overview panel shows real-time status.

-   **Open**: Click the port number to open the app in your browser.
-   **Resources**: Monitor CPU and RAM usage.

### Controls

| Button | Action |
| :--- | :--- |
| **Return to Logs** | View the console output. |
| **Stop** | Terminate the application. |
| **Reload** | Refresh the interface. |

![Started script](/images/started-script.png)

## 3. Stop

Click **Stop** in the overview panel or log view.
