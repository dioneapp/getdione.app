# Common Issues

If you encounter problems, check the solutions below or reach out to us.

## Quick Fixes

| Issue | Possible Solution |
| :--- | :--- |
| **Installation Stuck** | Check your internet connection and disk space. Try clicking **Stop** and restarting. |
| **App Won't Start** | The port might be in use. Restart Dione or your computer. |
| **Login Failed** | Ensure your default browser is supported. Try a different login method. |

## Reporting a Bug

If the issue persists, please report it with your **logs**.

-   [**Discord Community**](https://discord.gg/dioneapp): Get help from the community.
-   [**GitHub Issues**](https://github.com/dioneapp/dione/issues): Report technical bugs.
